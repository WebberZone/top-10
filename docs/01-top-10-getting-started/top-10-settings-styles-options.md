The **Styles** section contains a set of options that allow you to choose between various styles to customize the appearance of the popular posts list. These are global settings and many of these can be overridden in the widget or the shortcode.

## Popular posts style

Choose between the following styles in which the popular posts are displayed in the frontend.

### No styles

Select this option if you don't want the plugin to add any styles but respect the display of the thumbnails. You will need to add your own styles in this case

### Text only

This forces the display of the posts to be text only. i.e. no thumbnails

### Left thumbnails

This is the default style included within the plugin as displayed in the below screenshot.

## Custom CSS

You can add your own CSS here to style the output. Do not include `style` tags. The plugin will not fix your CSS code, so please ensure you included accurate code only.
